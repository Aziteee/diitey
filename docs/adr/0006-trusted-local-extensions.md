# 主题与插件是可信本地代码

主题和插件以站点所有者明确选择的完全可信代码运行在同一 Bun 进程中，职责限制是受支持接口与兼容性边界，而不是安全沙箱。相比隔离进程、能力沙箱或远程扩展市场，这允许直接使用 TypeScript 生态并显著降低 V1 复杂度；代价是恶意扩展可以访问进程资源，绕过接口的实现不受兼容性保证，站点不得加载不可信扩展。

**代码可信不等于访问者可信**。浏览器访问者必须通过 admin 会话鉴权才能使用管理 surface（ADR-0013）。插件归属（plugin ID、本插件服务与 admin Action）提供命名、数据所有权和兼容性约束，不声称能隔离恶意插件或构成插件间安全沙箱。
